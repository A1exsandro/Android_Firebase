# Android Firebase

##### An app for tasks and organization

### Some Features:
#### Todo
* A screen where you can add, edit and remove tasks.
* Will show a lista of tasks.

#### Doing
* Tasks added on "Todo", can be updated to "doing"
* Will show a lista of tasks.

#### Done
* Tasks can be upgraded to done.
* Will show a lista of tasks.

### Integration with firebase using the features:
* Authntication
* Realtime Database
* Analytics
